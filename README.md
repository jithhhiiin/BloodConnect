# BloodConnect

A full-stack blood donation management platform built using React.js, Django, and Django REST Framework.

## Features

- User registration and authentication
- Donor search based on blood group and location
- Emergency blood request system
- Donor request acceptance workflow
- Responsive user interface

## Tech Stack

### Frontend
- React.js
- Axios
- CSS

### Backend
- Django
- Django REST Framework
- SQLite

## Project Structure

Frontend:
React.js application

Backend:
Django REST API

## How to Run

### Frontend

npm install

npm run dev


### Backend

pip install -r requirements.txt

python manage.py runserver
